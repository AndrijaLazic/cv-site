<ContentBlock title="The Project">
  I worked on a web application designed to help non-technical employees manage their daily tasks more efficiently. This meant the backend had to be clear, reliable, and easy to support.

  <Section title="What the platform needed most">
    - Support for everyday task management.
    - Automatic and manual report generation.
    - Database auditing for better traceability.
    - Notifications for important events.
  </Section>
</ContentBlock>

<ContentBlock title="What I Built">
